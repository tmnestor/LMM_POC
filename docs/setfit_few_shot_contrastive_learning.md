# SetFit: Few-Shot Contrastive Learning

SetFit (Sentence Transformer Fine-Tuning) is a two-phase few-shot classification framework by Hugging Face and Intel Labs (2022). In Phase 1, a small set of labeled examples (as few as 8 per class) is combinatorially expanded into positive and negative sentence pairs, and a pre-trained sentence transformer is fine-tuned with contrastive loss to push same-class embeddings closer and different-class embeddings apart. In Phase 2, a logistic regression head is trained on the refined embeddings. This pair-generation step is sometimes called "self-supervised" because the pair labels are derived automatically from class labels, but the correct characterization is few-shot contrastive learning — it requires labeled data and does not use masking or pretext tasks on unlabeled corpora. SetFit achieves accuracy competitive with GPT-3 on few-shot benchmarks while using only an 80M-parameter model and minutes of training time (Tunstall et al., 2022, arXiv:2209.11055).
